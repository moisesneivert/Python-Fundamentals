# Agenda de Contatos em Python

Aplicação simples em terminal desenvolvida em Python para gerenciamento de contatos.

Este projeto foi criado com foco em demonstrar fundamentos da linguagem, organização de código e boas práticas de desenvolvimento.

---

## Funcionalidades

- Adicionar contatos
- Listar contatos
- Editar contatos
- Remover contatos
- Marcar e desmarcar contatos como favoritos
- Visualizar apenas contatos favoritos

---

## Tecnologias Utilizadas

- Python 3.10+
- Execução em terminal (CLI)
